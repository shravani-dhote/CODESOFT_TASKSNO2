✅ TaskFlow – Professional To-Do List

TaskFlow is a modern, responsive, and user-friendly
To-Do List web application that helps users organize daily tasks efficiently.
It includes task priorities, due dates, search, filtering, progress tracking, and task statistics.

 📌 Features

* ➕ Add new tasks
* 📝 Edit existing tasks
* 🗑️ Delete tasks
* ✅ Mark tasks as completed
* 🔴🟡🟢 Set task priority (High, Medium, Low)
* 📅 Add due dates
* 🔍 Search tasks instantly
* 🎯 Filter tasks

  * All
  * Pending
  * Completed
* 📊 Task statistics

  * Total Tasks
  * Completed Tasks
  * Pending Tasks
* 📈 Progress bar with completion percentage
* 💾 Local Storage support (tasks remain after refreshing)
* 📱 Fully responsive design

 🛠️ Technologies Used
